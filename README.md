# Résumé sur la Pérovskite

Une application Flutter présentant un résumé détaillé sur les pérovskites, leurs propriétés, et leur potentiel révolutionnaire dans le domaine de l'énergie solaire.

## Fonctionnalités
- Présentation générale des pérovskites.
- Explication des propriétés cristallines et optoélectroniques.
- Section dédiée aux applications (panneaux solaires, LED, etc.).
- Interface utilisateur claire et adaptative.

## Technologies
- Flutter

## Instructions pour l'exécution
1. Assurez-vous que Flutter est installé.
2. Exécutez `flutter pub get` pour installer les dépendances.
3. Lancez l'application avec `flutter run`.
